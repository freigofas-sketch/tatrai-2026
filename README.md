# Tatrų gidas

Offline PWA kelionės gidas šeimai po Tatrus. Gyvas adresas: https://freigofas-sketch.github.io/tatrai/

**Įsidiegti telefone:** iPhone — atidaryk adresą Safari → Bendrinti (Share) → „Įtraukti į pradžios ekraną". Android — atidaryk Chrome → meniu (⋮) → „Įdiegti programą" / „Įtraukti į pradžios ekraną".

> Projektas privatus — paslėptas nuo paieškos sistemų (`noindex, nofollow`).
